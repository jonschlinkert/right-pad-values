## Usage

**object**

```js
pad({
  a: 'b', 
  c: 'dddddd', 
  e: 'fff', 
  g: 'hhhhh'
});
```

Results in:

```js
{
  a: 'b     ',
  c: 'dddddd',
  e: 'fff   ',
  g: 'hhhhh ',
};
```

**array of objects**

Pass an array and specify the property with the value to pad:

```js
var pad = require('{%= name %}');

var arr = [{a: 'b'}, {a: 'bb'}, {a: 'bbbb'}, {a: 'bbb'}, {a: 'bb'}];
pad(arr, 'a');
```

Results in:

```js
[
  {a: '   b'},
  {a: '  bb'},
  {a: 'bbbb'},
  {a: ' bbb'},
  {a: '  bb'}
];
```
